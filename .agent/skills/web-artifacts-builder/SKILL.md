# Web Artifacts Builder

## Overview
Build complex, multi-component HTML artifacts for the agent.ai.

## Technical Stack
- React
- Tailwind CSS
- shadcn/ui components
- Lucide React icons

## Structure
- Single-file HTML execution
- State management via React
- Dynamic routing (if needed)

## Best Practices
- Keep components focused and modular.
- Use shadcn/ui for consistent, high-quality UI.
- Optimize for fast initial load.
