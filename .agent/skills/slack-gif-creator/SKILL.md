# Slack GIF Creator

## Overview
Create high-quality, lightweight animated GIFs optimized for Slack.

## Constraints
- **File size**: Max 1MB (Slack's limit for auto-play).
- **Format**: GIF.
- **Dimensions**: Square or 16:9 works best.

## Strategy
1. Use minimal frame counts for high-impact movement.
2. Optimize color palettes for small file sizes.
3. Focus on clear, recognizable actions.
